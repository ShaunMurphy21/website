PROOF OF CONCEPT

Dynamically generated map, generated markers and generated info about a crime commited. Info is scrapped from police API. Partial post code search will come with a later update.

Example usage: NG52SD or, with a specific date(error prone due to the API not deciding if it wants to respond with a 502 or empty array if no crimes :D) NG52SD 2022-02 - On testing, I believe the police API only holds data starting from 2021, so try there first.

Inaccuracies are possible and so are queriers with no results. I have not had the time to implement validation on queriers and responses from APIs. It is quite possible you'll get a map with no markers - or no map at all! I coded this in less than 5 hours starting at 2am. I'm tired :P
